Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) is a way of organizing and representing the elements of a web document (like HTML or XML) as a structured set of objects.

How do you select an element from the DOM?

The most common ways to select DOM elements:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.
    It allows to attach a single listener to the parent container.

    How do you manipulate an element's attributes and styles using the DOM?

    Manipulating Element's Attributes:

  • getAttribute(attributeName)
  • setAttribute(attributeName, value)
  • removeAttribute(attributeName)

    • Manipulating Element's Styles:
  • element.style.property
  • classList.add('className')
  • classList.remove('className')
